# Riconoscimento di persone
La progetto tende al riconoscimento di persone attraverso una webcam e l'utilizzo della librearia TensorFlow.

### Dataset di immagini utilizzati
- https://www.kaggle.com/datasets/constantinwerner/human-detection-dataset
- https://www.kaggle.com/datasets/ahmadahmadzada/images2000
- https://www.kaggle.com/datasets/alessiocorrado99/animals10
- http://vision.stanford.edu/Datasets/40actions.html
- http://vis-www.cs.umass.edu/lfw/#download
